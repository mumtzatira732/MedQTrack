**Clinic Queue Management System (Web & Mobile Application)**

MediQTrack is a clinic queue management system designed to improve patient flow, reduce waiting time uncertainty, and enhance communication between clinics and patients.  
The system consists of a **web-based backend and admin system** and an **Android mobile application** for patients.

This project was developed as an academic Final Year Project (FYP).

MediQTrack allows patients to:

Clinic staff and administrators can:
- Manage patient queues
- Monitor clinic operations through the admin dashboard
- View queue analytics and reports
